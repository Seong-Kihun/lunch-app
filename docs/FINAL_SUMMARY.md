# 🎉 Lunch App 개발 완료 요약

## 📊 프로젝트 개요

**Lunch App**은 직장인들의 점심 약속을 쉽게 만들고 관리할 수 있는 모바일 앱입니다.

### **기술 스택**
- **프론트엔드**: React Native + Expo
- **백엔드**: Flask + SQLAlchemy
- **데이터베이스**: SQLite (개발) / PostgreSQL (프로덕션)
- **캐시**: Redis (선택사항)
- **인증**: JWT + Magic Link
